# SnackShack Frontend

Web frontend for SnackShack — school café ordering platform.

## Features
- Online ordering
- Admin panel interface
- Responsive design
- Interactive menu

## Tech Stack
- HTML, CSS, JavaScript
- React
- Socket.IO (for real-time updates)


## Database

- MongoDB (Mongoose)
- Schemas:
  - User
  - Order
  - Product
  - Review
